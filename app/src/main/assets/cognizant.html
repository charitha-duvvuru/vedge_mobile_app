1.Problem Statement – Write a program to calculate the fuel consumption of your truck.The program should ask the user to enter the quantity of diesel to fill up the tank and the distance covered till the tank goes dry.Calculate the fuel consumption and display it in the format (liters per 100 kilometers).

Convert the same result to the U.S. style of miles per gallon and display the result. If the quantity or distance is zero or negative display ” is an Invalid Input”.

[Note: The US approach of fuel consumption calculation (distance / fuel) is the inverse of the European approach (fuel / distance ). Also note that 1 kilometer is 0.6214 miles, and 1 liter is 0.2642 gallons.]

The result should be with two decimal place.To get two decimal place refer the below-mentioned print statement :

float cost=670.23;

System.out.printf(“You need a sum of Rs.%.2f to cover the trip”,cost);

Sample Input 1:

  • Enter the no of liters to fill the tank

           20

  • Enter the distance covered

           150

Sample Output 1:

  • Liters/100KM

          13.33

  • Miles/gallons

          17.64

Explanation: 

  • For 150 KM fuel consumption is 20 liters, 
  • Then for 100 KM fuel consumption would be (20/150)*100=13.33,
  • Distance is given in KM, we have to convert it to miles (150*0.6214)=93.21,
  • Fuel consumption is given in liters, we have to convert it to gallons (20*0.2642)=5.284, 
  • Then find (miles/gallons)=(93.21/5.284)=17.64

Sample Input 2:

  • Enter the no of liters to fill the tank

         -5

Sample Output 2:

  • -5 is an Invalid Input

Sample Input 3:

  • Enter the no of liters to fill the tank

           25

  • Enter the distance covered

          -21

Sample Output 3:

  • -21 is an Invalid Input

Function calculateFuelConsumption():

    // Prompt user to enter the quantity of diesel and distance covered

    Print "Enter the no of liters to fill the tank"

    Read liters

 

    Print "Enter the distance covered"

    Read distance

 

    // Check if inputs are valid

    if liters <= 0 or distance <= 0:

        Print "Invalid Input"

        return

 

    // Calculate fuel consumption in liters per 100 kilometers

    consumption_liters_per_100_km = (liters / distance) * 100

 

    // Calculate fuel consumption in miles per gallon

    consumption_miles_per_gallon = (distance * 0.6214) / (liters * 0.2642)

 

    // Print the results with two decimal places

    Print "Liters/100KM"

    Print consumption_liters_per_100_km rounded to 2 decimal places

 

    Print "Miles/gallons"

    Print consumption_miles_per_gallon rounded to 2 decimal places

 

// Call the function to start the program

calculateFuelConsumption()

Function findMaxPlacementDepartment():

5.Problem Statement  – In a theater, there is a discount scheme announced where one gets a 10% discount on the total cost of tickets when there is a bulk booking of more than 20 tickets, and a discount of 2% on the total cost of tickets if a special coupon card is submitted. Develop a program to find the total cost as per the scheme. The cost of the k class ticket is Rs.75 and q class is Rs.150. Refreshments can also be opted by paying an additional of Rs. 50 per member.

Hint: k and q and You have to book minimum of 5 tickets and maximum of 40 at a time. If fails display “Minimum of 5 and Maximum of 40 Tickets”.  If circle is given a value other than ‘k’ or ‘q’ the output should be “Invalid Input”.

The ticket cost should be printed exactly to two decimal places.

Function calculateTotalCost():
// Prompt the user to enter the number of tickets
Print "Enter the no of ticket:"
Read numOfTickets

// Check if the number of tickets is within the valid range
if numOfTickets < 5 or numOfTickets > 40:
Print "Minimum of 5 and Maximum of 40 Tickets"
return

// Prompt the user if they want refreshments
Print "Do you want refreshment:y/n"
Read refreshmentOption

// Prompt the user if they have a coupon code
Print "Do you have coupon code:y/n"
Read couponOption

// Prompt the user to enter the circle
Print "Enter the circle:k/q"
Read circleOption

// Calculate the ticket cost based on the circle
if circleOption is not equal to 'k' and circleOption is not equal to 'q':
Print "Invalid Input"
return
else if circleOption is 'k':
ticketCost = 75.0
else:
ticketCost = 150.0

// Calculate the total cost of tickets without considering any discounts
totalCost = numOfTickets * ticketCost

// Apply discounts if applicable
if numOfTickets > 20:
totalCost = totalCost - (0.1 * totalCost)
if couponOption is 'y':
totalCost = totalCost - (0.02 * totalCost)

// Add refreshment cost if opted
if refreshmentOption is 'y':
totalCost = totalCost + (50 * numOfTickets)

// Print the total cost rounded to two decimal places
Print "Ticket cost:" + totalCost rounded to 2 decimal places
