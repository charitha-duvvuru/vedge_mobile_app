Function countNamesCut(N, ranks):

    Initialize a variable to store the count of names cut (cutCount) as 0

    Initialize a variable to store the previous rank (prevRank) as infinity

 

    for each rank in ranks:

        if rank < prevRank:

            Increment cutCount

        Set prevRank to rank

 

    Return cutCount

 

Input:

Read N from the user

Read ranks from the user as an array

 

// Call the function to count names cut

cutCount = countNamesCut(N, ranks)

Function countNumbers(n, k):

    Initialize a variable count to 0

 

    for i from 0 to n:

        if countSetBits(i) == k:

            Increment count

 

    Return count

 

Function countSetBits(num):

    Initialize a variable setBits to 0

 

    while num is not equal to 0:

        if num AND 1 is equal to 1:

            Increment setBits

        Right shift num by 1

 

    Return setBits

 

Input:

Read n, k from the user

 

// Call the function to count numbers with exactly k set bits less than n

count = countNumbers(n, k)

Function calculateMomentum(n, linkedList):

    Initialize momentum as 0

   

    // Traverse the linked list

    currentNode = linkedList.head

    while currentNode is not null:

        mass = currentNode.mass

        velocity = currentNode.velocity

       

        // Calculate momentum for current node and add it to the total momentum

        momentum = momentum + (mass * velocity)

       

        // Move to the next node

        currentNode = currentNode.next

   

    Return momentum

    Initialize Node:

        Initialize mass as 0

        Initialize velocity as 0

        Initialize next as null

5.Problem Statement : Anish is the laziest person you can ever see. He is tasked to write the name of the winner in a game where two people take part. And he just writes the longest common subsequence over there, so that with minimum chane or no backspace he can edit the name to the winner’s name.
For two given names, you have to predict what Anish will write in his computer before the start of the name. If there are more than two longest subsequences possible,write the one with less lexicographic value.
Input Format:
Two lines including two strings of name(All with capital letters)

Output Format:
A single line with the lexicographically smallest possible longest common subsequence.

Sample Input:
ABCD
BACD
Sample Output:
ACD
Explanation:
ACD and BCD these are the two possible biggest substring

Function longestCommonSubsequence(name1, name2):

    Initialize a 2D array dp of size (n1+1) x (n2+1) where n1 is the length of name1 and n2 is the length of name2

    Initialize dp[i][0] and dp[0][j] as 0 for all i and j

    Initialize a 2D array direction of size (n1+1) x (n2+1) to store the direction of movement

   

    for i from 1 to n1:

        for j from 1 to n2:

            if name1[i-1] is equal to name2[j-1]:

                Set dp[i][j] = dp[i-1][j-1] + 1

                Set direction[i][j] = "diagonal"

            else:

                if dp[i-1][j] >= dp[i][j-1]:

                    Set dp[i][j] = dp[i-1][j]

                    Set direction[i][j] = "up"

                else:

                    Set dp[i][j] = dp[i][j-1]

                    Set direction[i][j] = "left"

   

    Initialize an empty string result

    Set i = n1, j = n2

    while i > 0 and j > 0:

        if direction[i][j] is equal to "diagonal":

            Append name1[i-1] to result

            Decrement both i and j

        else if direction[i][j] is equal to "up":

            Decrement i

        else:

            Decrement j

   

    Return the reverse of result

 

Input:

Read name1 from the user

Read name2 from the user

 

// Call the function to find the longest common subsequence

result = longestCommonSubsequence(name1, name2)

 

// Print the result

Print result
