Function findHCF(a, b):

    while b is not equal to 0:

        temp = b

        b = a modulo b

        a = temp

    return a

2.Consider a string, S, that is a series of characters, each followed by its frequency as an integer. The string is not compressed correctly, so there may be multiple occurrences of the same character. A properly compressed string will consist of one instance of each character in alphabetical order followed by the total count of that character within the string.

×

Function compressString(S):

    Initialize an empty string result

 

    Sort the characters of S in alphabetical order

 

    i = 0

    while i < length(S):

        character = S[i]

        frequency = 0

 

        // Count the frequency of the current character

        while i < length(S) and S[i] is a digit:

            frequency = frequency * 10 + (S[i] - '0')

            i = i + 1

 

        // Append the character and its frequency to the result string

        result = result + character + frequency

 

    return result

Function findOriginalArray(pref[], n):

    Initialize an array arr of size n

    arr[0] = pref[0]  // The first element is the same as the prefix XOR

 

    for i from 1 to n-1:

        arr[i] = pref[i] XOR pref[i-1]  // Compute the original element using the prefix XOR

 

    return arr
